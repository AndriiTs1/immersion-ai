# IntegrationAI — V3 Vision

Этот документ фиксирует дальнюю цель продукта — полноценную платформу для множества
пользователей с AI-собеседником и системой прогресса. Это **не план на сейчас**.

V1 и V2 (см. `docs/CONCEPT.md`) должны сначала подтвердить гипотезу:
"Если каждый день проговаривать собственные мысли на итальянском, человек начинает
говорить быстрее". Эта гипотеза проверяется на одном пользователе (себе), без
бэкенда, без AI-диалогов, без авторизации. Всё, что описано ниже, становится
осмысленным только после того, как V1 пройден на практике, а не только спроектирован.

---

## Основной принцип

Пользователь приходит не учить язык, а решить задачу: через 2–3 месяца свободно
поддерживать бытовой разговор на итальянском.

Интерфейс максимально простой: Главный экран → Сегодняшний разговор → Практика →
Результат. Никаких сложных меню и десятков разделов.

---

## Архитектура продукта

### 1. Онбординг (2 минуты)

Собираем:

- текущий уровень (A1, A2, B1)
- цель (путешествия / переезд / работа / отношения / экзамен)
- сколько минут в день

После этого строится персональный план — например, "Уровень A2, цель: путешествия,
20 минут в день" → дорожная карта на 90 дней.

### Главный экран

Минималистичный. "Сегодня": Разговор дня (10 мин), Повтор слов (5 мин),
Произношение (3 мин), AI-диалог (10 мин). Прогресс: "День 17 из 90".

---

## Структура обучения

### Модуль 1. Выживание в разговоре

Темы: знакомство, рассказ о себе, семья, работа, хобби, покупки, ресторан, отель,
транспорт, врач.

Каждая тема содержит:

1. Ключевые фразы (например: "Mi chiamo Marco.", "Lavoro come sviluppatore.",
   "Abito a Zurigo.")
2. Мини-словарь — 15–20 слов, не больше
3. Разговорный шаблон ("Chi sei? → Mi chiamo... Da dove vieni? → Vengo da...")
4. Практика голосом — AI оценивает произношение, скорость, ошибки

### Модуль 2. AI-Собеседник

Самая ценная функция. Пользователь общается только голосом. Сценарии: официант,
продавец, сосед, коллега, друг, арендодатель.

После разговора — разбор ошибок с улучшенными вариантами
(например: "❌ Io sono andare → ✅ Io vado").

### Модуль 3. Система повторения слов

Не классический словарь — только слова из реальных разговоров пользователя.
Spaced repetition: новое слово → через 1 день → 3 дня → 7 дней → 14 дней → 30 дней.

### Модуль 4. Тренажёр произношения

Слушай → Повтори → Сравни. AI показывает ударение, проблемные звуки, оценку
(например: "Pronuncia 82/100, проблема: gli, gn, r").

### Модуль 5. Реальные ситуации

Геймификация через миссии: неделя 1 — заказать кофе, неделя 2 — спросить дорогу,
неделя 3 — заселиться в отель, неделя 4 — познакомиться с новым человеком.
Сценарий проходится полностью голосом.

## Навигация приложения

Пять вкладок: Home (сегодняшний урок), Dialogs (все разговоры), Vocabulary (слова),
Progress (статистика), Profile (настройки).

---

## Архитектура Backend

### Сервисы

- **Auth Service** — регистрация, вход
- **Learning Service** — курсы, уроки, темы
- **Conversation Service** — AI-диалоги, сценарии
- **Speech Service** — Speech-to-Text, Text-to-Speech, оценка произношения
- **Progress Service** — XP, достижения, статистика
- **Recommendation Service** — что повторять, какой урок следующий, слабые места
  пользователя

### База данных

Основные сущности: Users, Courses, Lessons, Dialogs, Vocabulary, Reviews,
PronunciationScores, Progress, Achievements.

### AI

GPT (или Claude) для диалогов, Whisper/STT для распознавания речи, TTS для озвучки.

---

## Идеальная ежедневная сессия пользователя

1. Открывает приложение
2. Запускает разговор дня
3. 10 минут говорит с AI
4. Получает исправления
5. Повторяет новые слова
6. Завершает урок

Весь цикл — 15–25 минут, 70–80% времени пользователь говорит, а не читает правила.

---

## Когда возвращаться к этому документу

После того как V1 (30 дней, ты сам) и V2 (близкие) подтвердят, что ежедневная
разговорная практика на личных фразах реально работает — то есть после того, как
гипотеза проверена на живых людях, а не только спроектирована на бумаге.
